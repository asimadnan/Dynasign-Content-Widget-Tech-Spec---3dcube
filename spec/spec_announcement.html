Sample Dynasign Content Widget Tech Spec

Template Definition

This template displays a title and description text objects with a color overlay on top of the standard background. Users can enter their own text and upload an image for the standard background. The template should be able to dynamically adjust the position of objects according to the display zone size.

  1. Name: announcement
  2. Zones: capable of display in various horizontal, vertical and square zone sizes w:h ratio from 5:1 to 1:5
  3. Type: Pixi
  4. External libs:
