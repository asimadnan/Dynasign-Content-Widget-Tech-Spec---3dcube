Build Widget Editor Data JSON File

Data JSON File includes the test data for the widget editor and the player

Editor Test Data JSON File (data_[template name].json)

To test the widget editor and widget player, you can change the field variable values in the test data file.

Explore sample test data files:
data_announcement.json
data_weather.json
data_sample.json

Steps to Build

  • Copy from data_announcement.json, then edit to make your own test data file data_[template name].json
  • Define Template name: e.g. "template": "announcement"
  • widget section (required for all widgets): add all field variables inside the content section
  • widgetAsset section (required for widgets with media assets): add media assets for testing.
    • For asset id, use w[wid]-[templateName]-[fieldId], e.g. w100-announcement-background_image
  • feed section (required for widgets with feeds): add feed data for testing
  • feedAsset section (required for feeds with media assets): add feed assets for testing
